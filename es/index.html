Crow Translate

Un traductor simple y ligero que permite traducir y reproducir el texto de manera hablada utilizando Google, Yandex, Bing, LibreTranslate y Lingva.

Star

Instalar Código fuente

Versión más reciente

Características

Velocidad

El programa está escrito en C++ / Qt y consume solo ~20 MB de RAM.

Código abierto

Crow Translate está publicado bajo una licencia GPL v3, lo que significa que puedes utilizarlo y modificarlo de manera libre.

Muchos idiomas

Gracias a Google, Yandex y Bing, puedes traducir a 117 idiomas diferentes.

Interfaz para la línea de comandos

Puedes traducir texto directamente desde la terminal.

Selección / OCR

Traducir y decir texto desde la pantalla o selección

Multiplataforma

Disponible para Linux y Windows.

Descargar

Paquetes

Windows

Windows requiere Microsoft Visual C++ Redistributable 2019 para funcionar.

📦 Paquete Scoop

sudo scoop install crow-translate -g

Linux

Per rendere l’applicazione nativa e senza icone mancanti su un ambiente desktop non KDE, è necessario configurare lo stile delle applicazioni Qt. Questa operazione può essere eseguita utilizzando qt5ct o adwaita-qt5 o qtstyleplugins. Si prega di controllare la guida all’installazione appropriata per la propria distribuzione.

Arch Linux, Manjaro, Chakra, etc

📦 Versión estable en AUR

git clone https://aur.archlinux.org/crow-translate.git
cd crow-translate
makepkg -si

📦 Versión de Git en AUR

git clone https://aur.archlinux.org/crow-translate-git.git
cd crow-translate-git
makepkg -si

📦 Repositorio Chaotic-AUR

sudo pacman -S crow-translate

Ubuntu, Linux Mint, KDE Neon, etc

📦 Launchpad PPA

sudo add-apt-repository ppa:jonmagon/crow-translate
sudo apt update
sudo apt install crow-translate

Fedora

📦 Fedora Copr

sudo dnf copr enable carlis/crow-translate
sudo dnf install crow-translate

CentOS, RHEL

📦 Fedora Copr

sudo yum copr enable carlis/crow-translate
sudo yum install crow-translate

Mageia 7

📦 Repositorio BlogDrake

# Con urpmi
sudo urpmi urpmi.addmedia --wget --distrib http://ftp.blogdrake.net/mageia/mageia7/x86_64 # Or i586
sudo urpmi crow-translate

# O con dnf
sudo dnf config-manager --add-repo http://ftp.blogdrake.net/mageia/BDK.repo
sudo dnf install crow-translate

openSUSE Tumbleweed

📦 Repositorio Tumbleweed

sudo zypper install crow-translate

openSUSE Leap

📦 Open Build Service

Solus

📦 Repositorio Solus

sudo eopkg it crow-translate
